Detailed Description

The ViewFlipper class doesn't do much but it is vital. The standard android ViewFlipper crashes when switching the orientation of the application (manually or automatically). When extending the default ViewFlipper and using this in the layout, the orientation-switch bug is gone. Unclear why, because this class hardly does anything, but it works!
